Thème : Vibe Coding — Coder avec l'intelligence artificielle

Pourquoi ce sujet ?

Dans le cadre de ma formation BTS SIO SLAM, j'ai choisi de suivre l'évolution du Vibe Coding — l'utilisation de l'IA comme assistant de programmation. Ce sujet est lié à mes pratiques et à l'évolution du métier de développeur. Il répond à la compétence Développement professionnel.

Qu'est-ce que le Vibe Coding ?

Le Vibe Coding consiste à utiliser l'IA comme assistant de programmation. Le développeur se concentre sur l'intention, tandis que l'IA génère ou complète le code.

Comment ça fonctionne ?

Via des instructions en langage naturel, l'IA produit des composants, fonctions ou API. Le développeur garde le contrôle final.

Apports et limites

  • Gain de temps significatif (+40% selon GitHub)
  • Aide à l'apprentissage pour les juniors
  • Risque de dépendance à l'IA
  • Qualité variable du code généré
